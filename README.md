# Playwright Automation Testing Portfolio

## Project Overview

This project is an automated testing framework developed to demonstrate modern QA automation practices. It focuses on end-to-end (E2E) testing for web applications using **Playwright** and **TypeScript**. The primary goal is to showcase a scalable, maintainable, and efficient testing structure.

## Key Features

- **Page Object Model (POM):** Organized the code into reusable page classes to improve maintainability and readability.
- **Data-Driven Testing:** Utilized JSON files to manage test data (users and products), allowing for easy updates and multiple test scenarios.
- **Authentication Persistence:** Implemented a global setup to handle login once and reuse the authentication state across multiple tests, significantly reducing execution time.
- **Cross-Browser Testing:** Configured to run tests across multiple browsers, including Chromium, Firefox, and WebKit (Safari).
- **CI/CD Integration:** Fully integrated with **GitHub Actions** to automatically trigger test suites on every push or pull request.

## Tech Stack

- **Language:** TypeScript
- **Framework:** Playwright Test
- **CI/CD:** GitHub Actions
- **Version Control:** Git & GitHub
